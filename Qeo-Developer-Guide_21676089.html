About This Guide

These guides provide the information you need to develop Qeo-enables applications on the supported development platform of your choice.

Why Use Qeo in Your Applications?

Qeo is based on a data exchange Publish/Subscribe (or in Qeo terms Write/Read) system. It is designed to be platform and device independent. It is geared towards distributed applications and software. If you are creating applications that run on one device only and one platform only, Qeo is probably not the optimal choice. However, if you want to create distributed applications that requires communication between devices different devices on different platform e.g. make Android devices communicate flawlessly with iOS or Linux based devices, Qeo is what you need.
