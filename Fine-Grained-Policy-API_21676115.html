Purpose Of The API

Before continuing with this section make sure to familiarize yourself with the concepts explained in the Introduction to Qeo Security and the Security Management Procedures.  More specifically, familiarize yourself with the section on How to Define a Policy.  As described in these pages the Qeo Policy allows an administrator to define from whom a user is allowed to read and to whom a user is allowed to write.  The API's described below allow an application to further limit this set of peers.

Icon

The API's described below can only be used when the administrator has configured a fine-grained policy for the realm in which the application runs.
